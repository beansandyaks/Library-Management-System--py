# Library-Management-System--py
A library management system built using python in IDLE for academic project.

This library management system allows the user to borrow the books from the library's book list.
There is also provision for the user to return the book which has been previously borrowed. 
The quantity of books borrowed and returned changes with every borrow or return transaction.
A bill is generated in .txt file with the name of the borrower and returnee with the tranascation details.
